# ResumeB
A fully professional Resume and cover letter generating tool, with ATS friendly resumes
📄 ATS-Friendly Resume & Cover Letter Builder

A powerful, purely client-side web application designed to help you build, customize, and optimize your resume and cover letter for Applicant Tracking Systems (ATS). Runs entirely in your browser with zero backend or build tools required.

✨ Features

Dual-Mode Editor: Seamlessly toggle between building your Resume and writing your Cover Letter.

Real-Time Live Preview: See your document update instantly as you type.

Built-in ATS Scoring Engine: * Evaluates your resume out of 100 points based on industry-standard parsing logic (Naukri, USAJOBS, etc.).

Analyzes completeness, keyword volume, action verbs, and quantifiable metrics.

Job Description (JD) Matching: Paste a target JD to get a Semantic & Keyword match score directly tailored to the role you want.

Rich Template Library:

16 distinct ATS-optimized Resume templates (e.g., Harvard Business, Google Standard, Wall Street, Tech Startup).

4 professional Cover Letter templates.

Global Typography Controls: Easily swap font families (Arial, Helvetica, Garamond, etc.) and dynamically scale base font sizes to make sure your resume fits perfectly on one page.

Data Portability & Persistence: * Auto-saves to your browser's local storage so you never lose progress.

Export/Import your resume data as a .json file to maintain multiple versions for different job applications.

1-Click PDF Export: Perfectly styled CSS @media print rules ensure your resume downloads flawlessly without any UI clutter using your browser's native print-to-PDF engine.

🚀 How to Run Locally

Because this is a completely serverless, single-file application, running it is incredibly simple:

Clone or download this repository to your local machine.

Double-click the index.html file to open it in your preferred web browser (Chrome, Edge, Safari, Firefox).

Start building!

🌐 Deployment (GitHub Pages)

This project is perfectly suited for free hosting via GitHub Pages.

Upload index.html to a public GitHub repository.

Go to your repository Settings > Pages.

Under "Build and deployment", set the source branch to main (or master) and save.

Your resume builder will be live and accessible via a public URL in a few minutes!

🛠️ Tech Stack

HTML5

Vanilla JavaScript (No complex frameworks or build steps required)

Tailwind CSS (Loaded via CDN for rapid styling)

FontAwesome (Loaded via CDN for sleek UI icons)

📝 Usage Tips

Downloading PDF: When you click the "Download PDF" button, ensure your browser's print destination is set to "Save as PDF", Paper Size is set to "A4" or "Letter", and Background Graphics are enabled (optional but recommended for certain templates).

ATS Optimization: Always paste the Job Description of the role you are applying for in the "Target Job Description" box to see which critical keywords you might be missing!
